# Einstein AI Coding Agent

Local-first AI coding system combining voice interface, code editor, app builder, and computer automation.

Built with OpenClaw + OpenJarvis + Ollama on Mac Studio M3 Ultra.

## Quick Start

See [README.md](README.md) for complete documentation.

## Components

- 🎤 Voice Interface (Jarvis)
- 💻 AI Code Editor (Cursor-style)
- 🏗️ App Builder (Lovable-style)
- 🤖 Computer Use (Claude-style)

## Performance

- Simple scripts: 15 sec
- Full apps: 45 sec
- 90% local processing
- ~$15/month cost
